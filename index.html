Projects

  • Facial expression recognition with Keras

    Built and trained a convolutional neural network (CNN) in Keras from scratch to recognize facial expressions supported by Rhyme. The data consisted of 48x48 pixel grayscale images of faces. Images of faces were classified based on the emotion shown in the facial expression into one of seven categories (0=Angry, 1=Disgust, 2=Fear, 3=Happy, 4=Sad, 5=Surprise, 6=Neutral). Used OpenCV to automatically detect faces in images and draw bounding boxes around them. Model predictions were directed to a web interface to perform real-time facial expression recognition on video and image data using Flask.

  • Classification of fruits based on their freshness

    Built an Android app to classify fruits (mango and pear) on the basis of freshness of fruits . This was achieved using resnet-50 model and custom made model . A release apk of the same was deployed as Institute Technical Project under Tech@IITB. The models were built in pytorch framework. The target API level of the app was 28 and the minimum API level supported was 21 and the maximum API level supported was 29. The app was primarily built in Java .

  • XLR8

    As an activity under Tech@IITB constructed a bot to overcome obstacles along a given trajectory and complete the competition task. The bot was controlled by an android app via Bluetooth.
